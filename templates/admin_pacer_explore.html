Admin playground

Explore PACER

This playground runs PACER Case Locator (PCL) immediate case searches so you can inspect the index fields that come back. It is designed for exploration and debugging, not bulk collection.

What this does: runs PCL index searches (/cases/find) and shows the raw response, observed fields, receipts, and logs.
What this does NOT do: it does not fetch docket entries, download documents, or store PACER credentials in the browser.
Cost warning: each page fetch is a billable PACER search and receipts are returned by PACER. Defaults are capped to one page ({{ explore_page_size }} records) and hard-limited to {{ explore_cap_pages }} pages ({{ explore_max_records }} records) unless PCL_EXPLORE_MAX_PAGES is increased server-side.
